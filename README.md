# pygodot
Python and Cython bindings for the Godot script APIs

The goal of this project is to provide Cython and Python language support for the Godot extension development.

This project is built on top of [godot-cpp](https://github.com/GodotNativeTools/godot-cpp).

## Package contents

### godot
Cython NativeScript 1.1 API bindings (work in progress)

### godot_cpp
Cython interoperanility layer for C++ NativeScript 1.1 API  (work in progress)
